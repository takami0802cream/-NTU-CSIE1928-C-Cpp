# Robot Collision

Problem statement draft saved from the original notes:

- Input initial positions of robot A and robot B
- Input `M`, `N`, `Fa`, and `Fb`
- Robot A moves east by `M*N`, then north by `M*N`, repeating until `Fa/(M*N)`
- Robot B moves north by `M*N`, then east by `M*N`, repeating until `Fb/(M*N)`
- Print `robots explode at time T` if they meet
- Otherwise print `robots will not explode`
